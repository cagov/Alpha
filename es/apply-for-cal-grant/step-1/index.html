Solicitar una beca Cal Grant: Paso 1

Fecha límite para solicitar: Lunes, 2 de marzo de 2020, 11:59p.m.

Presentar una solicitud para ayuda financiera, si no lo ha hecho

Hay dos maneras de solicitar ayuda financiera que puede conducir a una beca Cal Grant. Presentar un solicitud de FAFSA o CADAA, si no lo ha hecho.

    Aplicar para la Solicitud Gratuita de Ayuda Federal para Estudiantes (FAFSA) si:

  • Tienen la ciudadanía de EE.UU.
  • O es un Residente Permanente
  • O no tienen la ciudadanía de EE.UU., pero usted es elegible para ayuda financiera del gobierno
  • O tiene un Visa T
Aplicar al FAFSA

    Solicita la aplicación del Dream Act de California (CADAA) si usted:

  • Es indocumentado
  • O esta debajo el programa Acción para Llegados en la Infancia (DACA) - válido o vencido
  • O tiene un U Visa
  • O tiene el Estatus de Protección Temporal (TPS)
  • O es o un estudiante elegible para el programa AB540
Aplicar al CADAA
